What is a Variable?

Think of a variable as a box where you can store things. Imagine you have a bunch of different colored marbles, and you want to keep them organized. You can use different boxes to store different colors. In programming, variables are these "boxes" that store different types of data.

Creating a Variable

In Python, creating a variable is as easy as naming it and giving it a value. Here's how you do it (you can change box to whatever, it is just a placeholder, as well as the value it holds, this is just an example):

box = 10

Now, if you've ever coded before in a statically typed language, this might be confusing. Why don't I have to specify what type of variable box is? (int, str, float) Python is a dynamically typed language, and therefore the variable assignment is given at runtime, line by line.

this part of what makes python special. But this still doesn't mean Python is a magician, it can't determine what you want from thin air. that's where datatypes come in.

string

a string in python, or a str, is a piece of text inside that box. It can hold any data. It is shown with either a single quote ' to start and a ' to end, or a double quote so: 

box = "I am 1 box, I hold data!"
   #or
   box = 'I am 1 box, I hold data!'

Now if you notice, the variable holds numbers, letters, AND symbols! That's sick!

Integer

an integer is a type of variable that can only hold... guess... Integers. Now if you're a math nerd you know that an integer is a number, but not just any number, it has to be a whole number

box = 10

integer variables can only hold integers. Trying to give them a string will break them. They also do not use quotes.

Booleen

a Booleen is like a light switch, it can be on or off. It is useful for evaluating a statement's validity, and holding that answer. It uses True, or False (case sensitive) an example of a booleen is: 

box = True 
         box = False

There are many more uses for variables in python, and while I wont tell all of them in this lesson here are a few that you will learn. print(variable) will print the contents of the variable. if variable == True/False/Any_data: will check if the variable equals some data, and execute any code inside the if statement if true, or skip if false.
