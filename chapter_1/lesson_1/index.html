What is a Variable?

Think of a variable as a box where you can store things. Imagine you have a bunch of different colored marbles, and you want to keep them organized. You can use different boxes to store different colors. In programming, variables are these "boxes" that store different types of data.

Creating a Variable

In Python, creating a variable is as easy as naming it and giving it a value. Here's how you do it (you can change box to whatever, it is just a placeholder, as well as the value it holds, this is just an example):

box = 10

Now, if you've ever coded before in a statically typed language, this might be confusing. Why don't I have to specify what type of variable box is? (int, str, float) Python is a dynamically typed language, and therefore the variable assignment is given at runtime, line by line.
