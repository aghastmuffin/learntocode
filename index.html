Now let's begin.

Starting off simple. The "print" command prints text to screen. It's like asking a printer to print a page, but your screen is that page, and python is the ctrl+p! Nifty, right!

Python expects all arguments given to a function, or command, which in this case is print to be structured like this function(**args, **kwargs), with sometimes set types and sometimes unset types. This is pretty complicated for a beginner so for now let's ignore this and focus back on the basics. What this means is that print, the function, can take as many **args or "arguments" as needed. an "argument" is a special perameter passed to print, which supplies data to the function. Hello world is the data, and print is the instruction. Let's see it in action: 

print("hello, world!")
